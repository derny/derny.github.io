Angelic

Angelic is an actively developed Angular 1.3.x application boilerplate built on top of Bootstrap Sass that incorporates all of the features you would normally need when building a non-trivial SPA. Cross-cutting components such as authentication, pub/sub messaging and logging will all be included.

Continuous integration of UI builds with Jenkins has become a necessity in today's software development world and Angelic will have you covered with a GruntFile tailored to work with Jenkins along with backend configurations to get you up an running quickly.

If this all sounds of interest to you then stay tuned, Angelic is still a work-in-progress and will be ready near the end of January.
